==說明==

本漢化包是以「塞爱维&天邈文明5汉化包」為基礎所製作。
以 opencc (http://code.google.com/p/opencc/) 作為繁簡轉換工具。

會想要動手製作這個漢化包，一開始是因為原漢化包的安裝程式只能在簡體環境執
行，在正體環境下必須透過 AppLocale 才能安裝；此外，安裝好後若要轉成正體
中文，還必須額外下載檔案覆蓋，或是使用 Convertz 一個個資料夾進行轉碼。為
了方便正體中文的使用者，因而有了製作此漢化包的想法。

在製作過程中，遭遇了許多因修改失敗而使遊戲無法正常啟動的困難，但也因此對
於 Civilization V 的多國語系實作方式有了更深入的了解。因此後來決定直接作
成雙文字版本。除了嘉惠正體中文環境的使用者外，對於使用簡体中文環境，但對
正體中文有愛的朋友們相信也可帶來幫助。


==安裝與解除==

安裝：執行壓縮檔中 civ5zh_setup-0.9.exe 即可，請注意安裝過程中的遊戲路徑設定
      是否正確。安裝好進入遊戲後可從標題畫面進入以下選單切換語言：
      
      開始選單 > "OPTIONS(遊戲設置)" > "INTERFACE OPTIONS(介面選項)"
      
解除：執行遊戲路徑下的 civ5zh_uninstall.exe 即可。

本漢化包採用綠色安裝，在安裝與解除過程中，不會增刪修改登錄表／註冊表資料。
